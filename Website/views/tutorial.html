Tutorial

1. Create an account. Sign Up!

2. Download the starter package

Click here to download the starter package.(Latest Upload: 10:45am, 11th October, 2011)

3. Executing Sample bots

  1. Open terminal. Change your current directory to starterpackage/kkgame.
  2. To execute the sample bots(aa and bb) run ./game map1.txt aa bb.
    You can use any map present in starterpackage/maps.
  3. To see the game in action
    1. Change directory to starterpackage.
    2. Run ./view.
    3. Open viewer.html file in Mozilla Firefox or Google Chrome.

4. Testing your bot

  1. Make a folder containing your bot programs and the makefile.
    1. Name the makefile as "makefile".
    2. The executable name should be same as bot name.
    3. Name the folder same as executable name.
    4. Compress the folder. Use zip,tar or tar.gz. The compressed file name should also be same as executable name.
    Example:Suppose your bot's name is "my_bot", then the makefile should create a executable called my_bot. The folder and the compressed folder name should be my_bot and my_bot.zip respectively.
  2. Place the compressed folder inside starterpackage/kkgame.
  3. Follow the same instructions as Executing Sample bots.
    In step 2, replace one of the bot names' with your bot name.
    Example: Run ./game map1.txt my_bot aa

5. Submit your code.The last date to upload or change bot is 23:59hrs,29 October 2011.

6. Click to go to download page
